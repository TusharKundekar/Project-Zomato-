# Project-Zomato
🍽️ Zomato data analytics project analyzing customer behavior, sales trends &amp; restaurant performance 📊 using SQL, Excel, Tableau &amp; Power BI 💻. Includes data modeling, DAX &amp; interactive dashboards 🚀 to deliver actionable insights and support data-driven decisions 📈.

# 🍽️ Zomato Business Insights: Customer Behavior & Sales Performance  

An end-to-end data analytics project focused on analyzing food delivery data to uncover customer ordering patterns, restaurant performance, and revenue trends using multiple BI tools and SQL.

---

## 📌 Short Description / Purpose  
The Zomato Business Insights project analyzes food delivery data to identify key trends in customer behavior, order patterns, and restaurant performance. By combining SQL, Excel, Tableau, and Power BI, this project transforms raw data into actionable insights for improving business decisions, customer experience, and revenue growth.

---

## 🛠️ Tech Stack  
- 💻 Power BI & Tableau – Interactive dashboards and visual storytelling  
- 📊 Microsoft Excel – Data cleaning, Pivot Tables, dashboards  
- 🛢️ MySQL – Data extraction, joins, aggregations  
- 📈 Data Modeling – Star Schema (Fact & Dimension tables)  

---

## 🚀 Key Features  
- 📊 Multi-Tool Dashboards (Excel, Tableau, Power BI)  
- 📉 Sales & Revenue Analysis  
- 🍴 Customer Behavior Insights  
- 🗺️ Location-Based Performance Analysis  
- 📅 Time-Series Trends

## 🎯 Problem Statement  
Food delivery platforms struggle to understand customer preferences, identify top-performing restaurants, and optimize revenue strategies.

---

## 🎯 Goal of the Project  
To build a complete analytics solution that enables data-driven decision-making and provides insights into customer behavior and business performance.
